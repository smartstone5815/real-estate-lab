# 🏢 부동산 투자 분석 Lab (Real Estate Investment Analysis Lab)

부동산 매물별 투자 수익률을 정밀하게 분석하고 시뮬레이션하는 도구입니다.

## 🚀 프로젝트 목표
- 다양한 부동산 매물(상가, 오피스텔, 아파트 등)의 수익 구조 모델링
- 세금, 대출 금리, 공실률 등을 반영한 실질 ROI(수익률) 산출
- 매수/보유/매도 단계별 현금 흐름 시뮬레이션

## 🛠 기술 스택
- **Language**: Python 3.x
- **Library**: Pandas, Numpy-financial (예정)
- **UI**: Streamlit (예정)
